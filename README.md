# Vancouver Food Programs Dashboard

## Usage

Follow these steps to run the dashboard locally on your machine:

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/UBC-MDS/DSCI-532_2026_22_Vancouver-Food-Programs
    ```
2. Navigate to the project directory:
   ```bash
   cd DSCI-532_2026_22_Vancouver-Food-Programs
   ```
3. Install the environment:
   ```bash
   conda env create -f environment.yml
   ```
4. Activate the environment:
   ```bash
    conda activate vancouver-food-dashboard
    ```
5. Run the app by executing:
   ```bash
   shiny run --reload src/app.py
   ```
6. Open your web browser and navigate to `http://localhost:8000` to view the dashboard.
